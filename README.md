## 📖 Project Overview

This project is a **data analysis dashboard built in Microsoft Excel** to visualize and interpret emissions data in a simple and interactive way.

The dashboard uses Excel features like **Pivot Tables, Charts, and Slicers** to transform raw data into meaningful insights. It is designed for quick analysis of trends, comparisons, and category-wise performance.

---

## 🎯 Objectives

* To clean and organize raw emissions data
* To analyze trends over multiple years
* To compare emissions across different categories
* To build an interactive dashboard for easy understanding

---

## 🛠️ Tools & Techniques Used

* **Microsoft Excel**
* **Pivot Tables** – for summarizing large datasets
* **Pivot Charts** – for dynamic visualization
* **Slicers** – for interactive filtering
* **Data Cleaning** – handling missing or inconsistent data

---

## 📂 Dataset Description

The dataset used in this project contains:

* **Year** – time period of data
* **Category/Type** – type of emission
* **Values** – emission measurements

Data was preprocessed to ensure:

* No missing values
* Correct data types
* Structured tabular format

---

## ⚙️ Project Workflow

1. **Data Collection**

   * Imported dataset into Excel

2. **Data Cleaning**

   * Removed duplicates
   * Handled missing values
   * Standardized formats

3. **Data Transformation**

   * Created Pivot Tables
   * Aggregated data by year and category

4. **Dashboard Creation**

   * Designed charts (Line, Bar, Pie)
   * Added slicers for filtering
   * Organized layout for clarity

---

## 📊 Dashboard Features

* 📈 **Trend Analysis**: Shows how emissions change over time
* 📊 **Category Comparison**: Compares emissions across categories
* 🎛️ **Interactive Filters**: Slicers for dynamic data exploration
* 📉 **Summary View**: Quick insights from aggregated data
